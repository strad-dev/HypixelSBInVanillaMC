# Hypixel SkyBlock in Vanilla Minecraft plugin
This is a plugin intended to extend the gameplay of survival Minecraft by bringing select items and enemies from Hypixel SkyBlock, plus some custom ones.

## Features
- Iconic Hypixel SkyBlock weapons like **Hyperion** and **Terminator** as direct upgrades to Netherite and other vanilla items
- Custom SkyBlock bosses like **Necron** and the **Voidgloom Seraph**
- Rare loot like **Necron's Handle** and **Judgement Core** used to craft the custom items
- Utility items like **Wand of Restoration** intended to ease earlygame progression
- Want a challenge?  Chug a Bad Omen bottle and activate Hard Mode!

<sup>Note: The developers are not responsible for any rage-induced destruction of hardware.</sup>

## Key Gameplay Changes
- 1.8 regen and armor mechanics
- Improvements to undesirable enchants (COUGH KNOCKBACK) to make them useful
- Endgame swords have no attack cooldown

## Disclaimers: 
* This plugin is not the entirety of Hypixel SkyBlock and does not incorporate any features from there except for the selected items and mobs.
* This plugin is a fan project by a SkyBlock sweat and is not affiliated with or endorsed by Hypixel.
* **This plugin is intended to be used without any other combat-related plugins. It has not been tested with other plugins (and probably won't be for the foreseeable future)**

# Install the plugin
Go to the [releases](https://github.com/strad-dev/HypixelSBInVanillaMC/releases) page and download the latest version.
Add it to your server by dragging it into the `plugins` folder once downloaded.

**REQUIRES SPIGOT OR PAPER**
