# Getting Started

## Requirements

- **Minecraft Server:** Paper or Spigot 1.21.11
- **Java:** 21 or newer

## Installation

1. Download the latest SkyBlock in Vanilla plugin JAR.
2. Place the JAR file in your server's `plugins/` folder.
3. Start (or restart) the server.

That's it. No configuration files to edit -- the plugin works out of the box.

## Obtaining Items

Items in SkyBlock in Vanilla are obtained through two main paths:

- **Crafting** -- Many items have custom crafting recipes using materials found through mining or boss drops.
- **Boss Drops** -- Defeat bosses to earn rare weapons, armor, and crafting ingredients.

## Testing Items

If you are a server operator, you can use the `/getopitems` command to instantly receive all custom items for testing purposes. This command requires OP permissions.
