# Hard Sadan (M6)

<div class="stat-line">HP: 2000 (Phase 4) | Multi-Phase Fight</div>

*Hard Mode variant of: [Sadan](../generic/sadan.md)*

## Entity Type

Zombie (Giant) -- multi-phase encounter

## Summoning

Use [Giant Zombie Flesh](../../items/summon-items/index.md) on a Zombie while having **BAD_OMEN**.

## Abilities

Hard Sadan is a complex multi-phase encounter, the hardest dungeon-style boss in the plugin.

### Phase 1: Dummy

An initial warmup phase.

### Phase 2: Terracotta Swarm

A swarm of Terracotta mobs flood the arena. Clear them to proceed.

### Phase 3: Four Giants

Four Giant bosses must be defeated simultaneously.

### Phase 4: Giant Sadan

The true fight begins. Giant Sadan has **2000 HP** and access to all of normal Sadan's attacks with increased damage and frequency.
