# Hard Mode Bosses

Hard Mode bosses spawn when the summoner has the **BAD_OMEN** potion effect. They are significantly harder versions of existing bosses with unique mechanics, more HP, and better drop rates for rare items.

To get Bad Omen, kill a raid captain (illager with a banner) without triggering a raid.
