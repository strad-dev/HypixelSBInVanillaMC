# Generic Bosses

Generic bosses are summoned by using a specific summoning item on a specific mob type. Each boss has unique mechanics and drop tables.

| Boss | HP | Mob Type | Summon Item |
|------|----|----------|-------------|
| [Revenant Horror](revenant-horror.md) | 100 | Zombie | [Atoned Flesh](../../items/summon-items/index.md) |
| [Tarantula Broodfather](tarantula-broodfather.md) | 100 | Spider | [Spider Relic](../../items/summon-items/index.md) |
| [Chickzilla](chickzilla.md) | 500 | Chicken | [Omega Egg](../../items/summon-items/index.md) |
| [Zealot](zealot.md) | 130 | Enderman | [Corrupt Pearl](../../items/summon-items/index.md) |
| [meloG norI](melog-nori.md) | 50 | Iron Golem | [Antimatter](../../items/summon-items/index.md) |
| [Sadan](sadan.md) | 400 | Zombie (Giant) | [Giant Zombie Flesh](../../items/summon-items/index.md) |
| [Voidgloom Seraph](voidgloom-seraph.md) | 250 | Enderman | [Superior Remnant](../../items/summon-items/index.md) |
| [Highly Infuriated Wither Skeleton](infuriated-wither-skeleton.md) | 100 | Wither Skeleton | Spawn Egg |

All generic bosses have [hard mode variants](../hard-mode/index.md) that spawn when the summoner has BAD_OMEN.
