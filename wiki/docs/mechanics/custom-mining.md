# Custom Mining

When mining with a Fortune-enchanted tool, ores have a chance to drop [refined materials](../items/ingredients/mining.md) in addition to their normal drops.

## Fortune Multiplier

<div class="stat-line"><strong>Formula:</strong> effective chance = base chance x (1 + 0.25 x Fortune level)</div>

| Fortune Level | Multiplier |
|---------------|------------|
| Fortune I | 1.25x |
| Fortune II | 1.50x |
| Fortune III | 1.75x |
| Fortune IV | 2.00x |
| Fortune V | 2.25x |

## Divan's Pickaxe

[Divan's Pickaxe](../items/tools/divan-pickaxe.md) **doubles all ore drops** and stacks with Fortune, making it the best mining tool in the plugin. It requires all refined ores plus an [Alloy](../items/ingredients/mining.md#other-mining-materials) to craft.
