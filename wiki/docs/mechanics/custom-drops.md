# Custom Drops

The plugin completely replaces vanilla Minecraft mob drops. Every mob type has a custom drop table.

## Looting Bonus

The Looting enchantment affects drop chances for RNG-based drops:

<div class="stat-line"><strong>Looting I:</strong> +15%</div>
<div class="stat-line"><strong>Looting II:</strong> +30%</div>
<div class="stat-line"><strong>Looting III:</strong> +50%</div>
<div class="stat-line"><strong>Looting IV:</strong> +70%</div>
<div class="stat-line"><strong>Looting V:</strong> +100% (requires <a href="../items/ingredients/misc.md#enchantment-upgrader">Enchantment Upgrader</a> to craft)</div>

For example, if an item has a 5% base drop chance and you have Looting III, the effective chance becomes 5% x 1.5 = 7.5%.
