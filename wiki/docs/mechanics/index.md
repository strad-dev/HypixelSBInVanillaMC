# Mechanics

SkyBlock in Vanilla overrides and extends many of Minecraft's core systems.

## Core Systems

<div class="stat-line"><strong><a href="damage-system/">Damage System</a></strong> -- Replaces vanilla damage calculations with a custom NMS-based system.</div>
<div class="stat-line"><strong><a href="mana-system/">Mana System</a></strong> -- The Intelligence system that powers <a href="item-abilities/">item abilities</a>.</div>
<div class="stat-line"><strong><a href="item-abilities/">Item Abilities</a></strong> -- How ability items like <a href="../items/weapons/hyperion/">Hyperion</a> and <a href="../items/tools/aspect-of-the-void/">AOTV</a> work.</div>

## Loot and Progression

<div class="stat-line"><strong><a href="custom-drops/">Custom Drops</a></strong> -- Custom drop tables for all mobs, with Looting scaling.</div>
<div class="stat-line"><strong><a href="custom-mining/">Custom Mining</a></strong> -- Fortune-based <a href="../items/ingredients/mining/">refined material</a> drops from ores.</div>
<div class="stat-line"><strong><a href="crafting-recipes/">Crafting Recipes</a></strong> -- All custom recipes for <a href="../items/weapons/">weapons</a>, <a href="../items/armor/">armor</a>, <a href="../items/tools/">tools</a>, and enchantments.</div>

## Difficulty

<div class="stat-line"><strong><a href="hard-mode/">Hard Mode</a></strong> -- BAD_OMEN triggers <a href="../bosses/hard-mode/">harder boss variants</a> with better drop rates.</div>
