# Commands

All available commands in SkyBlock in Vanilla.

## Item Commands

### `/getopitems [itemSet] <player>`

Gives a set of SkyBlock items to the specified player. **Requires OP permission.**

Valid item sets:
- `combat` -- combat-related items (weapons, armor).
- `ingredient` -- crafting ingredients and materials.
- `summon` -- boss summoning items.

## Communication Commands

### `/w [player] [message]`

Send a private message to a player. Aliases: `/tell`, `/msg`. All three commands do the same thing.

### `/say [message]`

Broadcast a message to all players on the server.

### `/me [action]`

Send an action message (e.g., `/me waves` displays "* PlayerName waves").

## Utility Commands

### `/locateplayer [player]`

Displays the coordinates and dimension of the specified player.

## Admin Commands

### `/resetwitherfight`

Resets the Wither Lords boss fight status. Use this command if the boss died unexpectedly or the fight got stuck in a broken state.

### `/m7tasactivatewitherfight`

Sets the Wither Lords fight to an active state. This is an admin/TAS utility command for debugging or testing purposes.
